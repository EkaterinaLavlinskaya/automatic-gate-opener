# 🚗 SmartGate: Система умного контроля доступа с распознаванием автомобилей

Это полностью рабочий прототип системы, которая:

- управляет воротами через компьютер (Arduino + реле);
- записывает видео с камеры;
- детектирует движение и начинает запись;
- распознает автомобили с помощью YOLO;
- дообучена на моем собственном датасете для распознавания машин спереди.

Проект написан на **Python** с использованием **OpenCV**, **YOLOv3**, **PyTorch** и **Arduino**.

## 🧠 Как я это делала (история проекта)

Этот проект — не про «скачал и запустил». Это путь длиной в несколько месяцев, с реальными вызовами:

1. **Python и OpenCV** —  написала скрипт для камеры.
2. **Железо** — подружила Python с Arduino и реле, чтобы открывать ворота командой.
3. **Детекция движения** — научила камеру писать только когда что-то происходит.
4. **YOLO** — запустила нейросеть на ноутбуке, научилась менять пороги и размер кадров.
5. **Проблема ракурса** — YOLO не видел машины спереди.
6. **Сбор датасета** — сделала 45 фото машин спереди (в разное время суток, с разного расстояния).
7. **Разметка** — вручную обвела машины в LabelImg, получила 45 txt-файлов.
8. **Дообучение** — запустила обучение YOLOv3 в Google Colab (GPU), получила свою модель.
9. **Итог** — модель теперь уверенно находит машины спереди.

## 📌 Business Context
Система для автоматического контроля доступа на парковку / КПП. Распознает номер автомобиля по изображению и проверяет его в базе разрешенных.

## 🎯 Key Features
- **Detection**: YOLOv8 для локализации номера
- **OCR**: EasyOCR (поддержка русского и английского)
- **Access control**: проверка по базе (CSV/SQLite)
- **API**: REST API на FastAPI
- **Docker**: контейнеризация для легкого развертывания
- **Tests**: юнит- и интеграционные тесты
